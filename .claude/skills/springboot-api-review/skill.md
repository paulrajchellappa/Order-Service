---
name: springboot-api-review
description: Review Java Spring Boot APIs for production readiness.
---

When reviewing Spring Boot APIs, check:

1. Controller validation
2. Service layer separation
3. Exception handling
4. Retry and timeout configuration
5. Kafka error handling
6. Logging with correlation ID
7. Prometheus metrics
8. Unit and integration tests
9. Security issues
10. Secrets exposure

Return:
- Findings
- Risk level
- Suggested fixes
- Code changes if requested
